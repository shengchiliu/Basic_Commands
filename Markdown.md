# General Markdown Commands

#### Header
```
# largest heading
## second largest heading
###### smallest heading
```

#### Styling Text
```
**Bold**
*Italic*
**_Bold and italic_**
~~Strikethrough~~
```

#### Comment
```
[comment]: <put comments here!>
```

#### Quoting text
> Put text here

#### Code Block
```
```
Put code here
```
```

#### Link
```
[website](https://www.website.com/)
```

#### Nested Lists
```
1. Item
   - 1st Nested Item
     - 2nd Nested Item
```
